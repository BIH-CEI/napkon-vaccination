Resource Profile: SARS CoV2 Infection

Defining URL:https://www.napkon.de/fhir/StructureDefinition/sars-cov2-infection
Version:1.0.0
Name:SARSCoV2Infection
Title:SARS CoV2 Infection
Status:Active as of 2021-11-30T16:36:32+00:00
Definition:

Profile to capture the diagnosis of disease caused by 2019 novel coronavirus.

Source Resource:XML / JSON / Turtle

The official URL for this profile is: 

https://www.napkon.de/fhir/StructureDefinition/sars-cov2-infection

Guidance

Parameter Case Report Form Response Options
Has the patient been infected with the novel coronavirus (SARS-CoV-2) in the past or has the patient tested positive? Yes / No / Unknown

To represent the presence, absence or uncertainty of findings, use Condition.verificationStatus with codes from ConditionVerificationStatus ValueSet and/or SNOMED CT codes from ConditionVerificationStatusSNOMED ValueSet or uncertaintyOfPresence on Condition.extension.

Response Optionoption verificationStatus uncertaintyOfPresence extension
Present ConditionVerificationStatus: #confirmed
SNOMED CT: Confirmed present (qualifier value)		 -
Absent ConditionVerificationStatus: #refuted
SNOMED CT: Definitely NOT present (qualifier value)		 -
Unknown - set

Note the additional constraint on Condition that checks if either verificationStatus or the uncertaintyOfPresence extension is present:
unk-1: If verificationStatus is present, uncertaintyOfPresence-Extension SHALL NOT be present. verificationStatus.exists() != Condition.modifierExtension.where(url = ‘https://www.netzwerk-universitaetsmedizin.de/fhir/StructureDefinition/uncertainty-of-presence’).exists()

This Condition profile is used in conjunction with the COVID-19 Encounter profile and the COVID-19 Oxygen Therapy profile to indicate the course of the infection.

​This profile of a FHIR Condition is derived from the diagnosis-covid-19 profile of the German Corona Consensus Dataset (GECCO).
